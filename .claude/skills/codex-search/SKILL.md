# Codex Search

## 概要

このスキルは、Claude Codeの標準的なWebSearchやWebFetchツールでは不十分な場合に、Codex MCPの高度なweb検索機能を活用します。より深く包括的な検索結果を提供し、構造化された形式で返します。

## 使用するタイミング

以下のシナリオでこのスキルを使用します:

1. **WebSearch/WebFetch失敗時**: 標準の検索ツールで情報が取得できない場合
2. **不十分な結果**: 繰り返し検索しても必要な情報が得られない場合
3. **明示的なリクエスト**: ユーザーが「codexで検索」や「codex検索」を依頼した場合
4. **不完全な情報**: 標準の検索結果が深さや最新性に欠ける場合

**重要**: まず標準のWebSearchを試してください。標準の方法で不十分な場合のみ、このスキルを使用してください。

## 動作方法

このスキルは`mcp__codex__codex`ツールを使用して:

1. Codexセッションで検索クエリを実行
2. 包括的な検索結果を取得
3. 以下の情報を含む構造化された出力を生成:
   - 検索結果の要約
   - 主要な情報ポイント
   - 関連URL

## 使用パターン

このスキルがトリガーされたら、以下のパターンに従って検索を実行してください:

```
mcp__codex__codexを使用し、以下のような明確なプロンプトを指定:
"[ユーザーの検索クエリ] について web検索して、以下の形式で結果を返してください:
- 概要: [検索結果の要約]
- 主要な情報: [箇条書き]
- 参照URL: [関連するURL一覧]"
```

## 期待される出力

Codexは以下を含む構造化された結果を返します:

- **概要**: 検索結果の要約
- **主要な情報**: 重要なポイントと詳細
- **参照URL**: 参考となる関連URLのリスト

Codexから結果を受け取った後、適切なコンテキストと共にユーザーにメイン会話で提示してください。

## 使用例

ユーザーリクエスト: "最新のReact 19の情報を探して"

アクション:
1. まず標準のWebSearchを試す
2. 不十分な場合、このスキルをトリガー
3. `mcp__codex__codex`を以下のプロンプトで呼び出す: "React 19の最新情報について web検索して、概要、主要な機能、参照URLを返してください"
4. Codexの構造化された結果をユーザーに提示

## 注意事項

- Codex検索は標準の検索より時間がかかる場合があります
- 結果は通常、より包括的で最新です
- 初回検索で得られたthreadIdを`mcp__codex__codex-reply`で再利用して、フォローアップクエリを実行できます
